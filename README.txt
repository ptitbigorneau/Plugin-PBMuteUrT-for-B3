# pbmuteurt plugin for UrT
# Plugin for B3 (www.bigbrotherbot.com)

# www.ptitbigorneau.fr

pbmuteurt plugin b3 (v1.2) for Urban Terror

Installation

------------


* copy pbmuteurt.py into b3/extplugins
 

* copy pbmuteurt.ini into b3/extplugins/conf
 

* create the pbmuteurt table in your database importing the pbmuteurt.sql file.
 

* update your main b3 config file with :
   <plugin name="pbmuteurt" config="@b3/extplugins/conf/pbmuteurt.ini"/>
